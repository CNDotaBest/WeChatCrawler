## WeChatCrawler

WeChatUrlCrawler是一个专门爬取公众号文章url的爬虫，将爬到的url保存在文件或者数据库中

#### 环境准备：

准备python3.6及以上的环境，安装需要的库，如果报错

`ModuleNotFoundError: No module named 'XXX'`,只需要执行`npm install XXX`即可

#### 具体使用步骤，可以搜索公众号”程序员修炼“，或者扫描下方二维码
